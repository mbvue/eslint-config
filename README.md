## Vendor
ESLint config.

js|jsx|ts|tsx|vue and prettier

## Installation
yarn add -D @mbvue/eslint-config

## Usage
.eslintrc.js：

module.exports = {
  extends: ["@mbvue/eslint-config"],
};

or

.eslintrc：

or

{
  extends: ["@mbvue/eslint-config"],
}

or

package.json：

{
    "eslintConfig": {
        "extends": "@mbvue/eslint-config"
    }
}
